# AdventureWorksCSV

This repository contains the CSV files for the AdventureWorks 2019 database.

## Data Folder Structure

```
data/
├── AWBuildVersion.csv
├── DatabaseLog.csv
├── HumanResources/
│   ├── Department.csv
│   └── ...
├── Person/
│   ├── Address.csv
│   └── ...
├── Production/
│   ├── BillOfMaterials.csv
│   └── ...
├── Purchasing/
│   ├── ProductVendor.csv
│   └── ...
└── Sales/
    ├── CountryRegionCurrency.csv
    └── ...
```
